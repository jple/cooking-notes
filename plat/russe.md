# Boeuf Stroganoff

- [pas mal détaillé](https://www.editions-larousse.fr/recette/boeuf-stroganov/)

## Mots clés
- Boeuf mijoté
- Boeuf à cuisson rapide
- Recette rapide
- Choix des morceaux de boeuf
    * (faux-)filet coupé en lanière fin
    * filet mignon, rumsteak
    * bavette

## Ingrédients

- boeuf
- champignon
- oignon, ail

- marinade
    * sauce soja, piment
    * yaourt
    * vin blanc
    * oignon, échalotte, ail

- sauce :
    - bouillon de boeuf
    - crème fraiche épaisse
    - moutarde

## Recette
La veille :
- couper le boeuf en filet / lanière / aiguillette de 
    - version 1 : 2,5cm
    - version 2 : 4cm d'épaisseur pour une viande saignante
- mariner le boeuf la veille
    * version 1 : vin blanc
    * version 2 : soja + piment (worcestershire), ou vinaigre balsamique
    * version X : yaourt, vinaigre

Le jour J
- faire dorer boeuf
- retirer
- faire revenir légume, oignons, ...
- ajouter bouillon, crème, moutarde, (farine pour épaissir)
    * version 1 : faire épaissir la sauce (10mn) avant de mettre le boeuf
    * version 2 : mettre le boeuf avant d'épaissir la sauce pour mijoter 30mn
    * version 3 : mettre boeuf + bouillon à cuire (10mn), puis crème à épaissir (10mn)
