# Do chua (classique)

- eau   : suffisamment pour recouvrir les carottes
- vinaigre  :   1/7 d'eau
- sucre     :   1/14 d'eau
- carotte   :   rapé

- faire bouillir le tout (sans les carottes)
- laisser refroidir

- mettre les carottes dans des bocaux
- remplir les bocaux avec 1cm au dessus des carottes

- manger au bout de 3-4 jours
